# Resort Management System

A desktop-based resort management system developed using Python (Tkinter) and MySQL to handle bookings, services, and billing.

## Features
- Customer and manager interfaces
- Room booking with date-based reservations
- Service selection (spa, transport, dining, etc.)
- Dynamic billing based on selected services
- Secure login and user authentication
- Manager access to customer bills using Booking ID

## Tech Stack
Python, Tkinter, MySQL
